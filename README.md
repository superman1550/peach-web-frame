

# Peach Web Frame

Peach Web Frame 是一个基于 Java 的轻量级 Web 应用框架，提供了丰富的 UI 组件、数据访问层和事件处理机制。

## 特性

- **丰富 UI 组件**: 内置多种 Web 界面组件，包括文本框、组合框、树形控件、数据表格、日历等
- **数据访问层**: 完整的 DAO 架构，支持数据库 CRUD 操作
- **事件驱动**: 完善的事件监听机制，支持动作、选择、变更等多种事件
- **布局管理**: 灵活的窗口布局系统，支持东南西北中五区域布局
- **配置管理**: 便捷的配置文件读取与系统参数管理

## 项目结构

```
cn/jpeach/frame/
├── action/          # 事件监听器
├── cache/           # 缓存管理
├── comp/            # UI 组件
├── conf/            # 配置管理
├── core/            # 核心框架
├── dao/             # 数据访问层
├── entity/          # 实体类
├── exception/       # 异常定义
└── util/            # 工具类
```
